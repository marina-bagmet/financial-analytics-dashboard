### 1. Общая информация
Дашборд представляет обзор ключевых метрик (выручка, дельта выручки, продажи, средний % выкупа, средняя цена товара). Основная цель - помочь менеджерам, аналитикам и руководителям смотреть динамику показателей и принимать на их основе решения.

Владелец дашборда: Багмет Марина Андреевна

Основной заказчик: Руководитель команды продукта

Метрики:

* Выручка

* Дельта выручки(прирост или отклонение в абсолютном и % значении)

* Продажи( с учетом отказов, в единицах товара).

* Средний % выкупа

* Средняя цена товара
